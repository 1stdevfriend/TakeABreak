# Take a break

## Introduction 
Batch script that notifies to take a break and sleeps computer after every 2 hours. 

## Based on 
- Batch Scripting

## Works on 
- Windows 7/8/8.1/10

## Installation
- Open install folder
- Double click on installer as per your OS architecture(32/64 Bits)
- Restart and Done! :) 

## Uninstallation
- Open uninstall folder
- Double click on installer as per your OS architecture(32/64 Bits)
- Restart and Done! :)
